# Vehicle Maintenance Scheduler API

Backend REST API project built using Node.js and Express.js.

## Features

- CRUD APIs
- Logging Middleware
- Service Layer Architecture
- Validation
- Error Handling
- RESTful API Design

---

## Technologies Used

- Node.js
- Express.js
- Axios
- Postman

---

## API Endpoints

### GET All Vehicles

GET /api/vehicles

---

### ADD Vehicle

POST /api/vehicles

---

### UPDATE Vehicle

PUT /api/vehicles/:id

---

### DELETE Vehicle

DELETE /api/vehicles/:id
